Installing This Spree Ember Addon

We recommend installing Spree Ember into your Ember CLI app by using the main addon, Spree Ember.

ember install:addon spree-ember

Then follow the prompts in the generator to install the correct packages for your use case.


However, if you know what you're doing, and you only want the functionality from this addon, you can install it seperately, instead.

ember install:addon ember-cli-spree-checkouts
